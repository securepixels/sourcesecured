## CyberSecurity Tools

- **[Kali Linux](https://www.kali.org/)**: A Debian-based Linux distribution designed for digital forensics and penetration testing. It comes pre-installed with numerous security tools and is widely used for ethical hacking.
- **[Metasploit](https://www.metasploit.com/)**: A powerful penetration testing framework that allows security professionals to find and exploit vulnerabilities in systems and applications.
- **[Wireshark](https://www.wireshark.org/)**: A network protocol analyzer that helps you capture and interactively browse the traffic on a computer network. It's essential for troubleshooting network issues and analyzing network protocols.
- **[Burp Suite](https://portswigger.net/burp)**: A popular tool for web application security testing, allowing users to identify vulnerabilities and perform security assessments of web applications.
- **[Nmap](https://nmap.org/)**: A network scanning tool that helps you discover hosts and services on a computer network, providing insights into the security of the network.
- **[OWASP ZAP (Zed Attack Proxy)](https://owasp.org/www-project-zap/)**: A free and open-source web application security scanner that is great for finding security vulnerabilities in web applications.
- **[CyberChef](https://gchq.github.io/CyberChef)**: Ciphers, Data manipulation and Analysis.
- **[ctf-tools](https://github.com/topics/ctf-tools)**: Repository of tools commonly used in CTF challenges.
- **[Ultimate List of SANS Cheat Sheets](https://www.sans.org/blog/the-ultimate-list-of-sans-cheat-sheets): Cheat sheets for quick reference on various cybersecurity topics.
- **[JSFiddle](https://jsfiddle.net/)**: Online editor for testing and running HTML, CSS, and JavaScript, useful for web exploitation tasks.
- **[Regexr](https://regexr.com/)**: Tool for learning, building, and testing regular expressions, which are useful for searching patterns in text.
- **[Postman](https://www.postman.com/)**: API development and testing tool for interacting with web services, often used in CTF web challenges.
- **[ZAP](https://www.zaproxy.org/)**: Tools for scanning web applications and identifying vulnerabilities, commonly used in web-based CTF challenges.
- **[PacketLife Cheat Sheets](https://packetlife.net/library/cheat-sheets)**: Network-related cheat sheets that provide quick references for networking tasks such as packet analysis.
- **[Guide for CTFs](https://picoctf.org/resources.html)**: A comprehensive resource page for CTF participants, providing useful guides and materials.
- **[CTF Frameworks, Libraries, Resources, and Software](https://github.com/apsdehal/awesome-ctf)**: Curated list of CTF resources, frameworks, and libraries for participants.
- **[Flaws.cloud](http://flaws.cloud/)**: A site dedicated to finding and exploiting vulnerabilities in web applications through interactive challenges.
- **[Flaws2.cloud](http://flaws2.cloud/)**: Similar to Flaws.cloud, offering additional challenges focused on security flaws in applications.
- **[Smash The Stack](https://www.smashthestack.org/main.html)**: A resource providing challenges and tutorials aimed at improving security and programming skills.
- **[MetaCTF](https://app.metactf.com/)**: A practice CTF environment with challenges and solutions from MetaCTFs's previous Flash CTF competitions.
- **[IDA Free](https://hex-rays.com/ida-free/)**: Free version of the Interactive DisAssembler, a powerful tool for reverse engineering.
- **[Cipher Identifier](https://www.dcode.fr/cipher-identifier)**: Online tool to identify various ciphers used in cryptography challenges.
- **[Decompiler Explorer](https://www.dcode.fr/cipher-identifier)**: Online tool to identify various ciphers used in cryptography challenges.
- **[Epoch Time Converter](https://www.epochconverter.com/)**: Online hex editor for viewing and editing binary files directly in the browser.
- **[Rockyou.txt Download](https://www.google.com/url?sa=t&source=web&rct=j&opi=89978449&url=https://github.com/brannondorsey/naive-hashcat/releases/download/data/rockyou.txt&ved=2ahUKEwjNveLLtLuIAxU8FVkFHaMqASwQFnoECAgQAQ&usg=AOvVaw3snAERl1mU6Ccr4WFEazBd)**: Link to download the famous Rockyou.txt password list, commonly used in password cracking exercises.
- **[Wordlists](https://github.com/danielmiessler/SecLists)**: It's a collection of multiple types of lists used during security assessments, collected in one place.
- **[Linux Upskill Challenge](https://linuxupskillchallenge.org/)**: A month-long course aimed at those who aspire to get Linux-related jobs in the industry
- **[Linux Journey](https://linuxjourney.com/)**: Learn the ways of Linux-fu, for free
- **[Posters & Cheat Sheets](https://www.sans.org/posters/)**: 12 pages of resources
- **[Ultimate Guide to Mastering Nmap and Netcat](https://hackernoon.com/the-ultimate-guide-to-mastering-nmap-and-netcat)**: In-depth guide covering the usage, commands, and ethical practices for Nmap and Netcat in network security.
- **[DFIR Diva Training](https://dfirdiva.com/training/)**: Training resources for digital forensics and incident response, with courses and webinars.
- **[Steganalysis Tool](https://www.aperisolve.com/)**: Online tool for analyzing images to uncover hidden messages and data.
- **[The DFIR Report](https://thedfirreport.com/)**: A website providing case studies and reports on digital forensics and incident response.
- **[Whitepapers](https://www.sans.org/white-papers/)**: Top-of-mind papers in the SANS community
- **[Capture the Flag Beginner Guide 2024](https://youtu.be/YAqZmMYudZU?si=d1fhD6FGR__5Tgi9)**: YouTube guide designed for beginners in Capture the Flag competitions, aimed at helping them become hackers.

## Virtual Machines

Virtualization platforms for running multiple operating systems in virtual environments, essential for CTF practice.

- **[VMware](https://www.vmware.com)**:  A powerful virtualization software that allows you to run multiple operating systems on a single physical machine, ideal for testing, development, and cybersecurity labs.
- **[VirtualBox](https://www.virtualbox.org)**: A free and open-source virtualization tool that supports various operating systems, making it a popular choice for building home labs and experimenting with virtual environments.
- **[Parallels](https://www.parallels.com)**: A virtualization solution optimized for Mac users, enabling seamless access to Windows and Linux systems on macOS, perfect for cross-platform testing and development.


Pre-configured virtual machines with security and forensic tools for use in various CTF environments.

- **[Kali Linux](https://www.kali.org/)**: A Linux distribution tailored for penetration testing and cybersecurity, offering a robust toolkit with pre-installed tools for ethical hacking, forensics, and vulnerability assessments.
- **[Ubuntu](https://ubuntu.com)**: A widely-used Linux distribution known for its ease of use and stability, ideal for development, server hosting, and as a base for cybersecurity environments.
