## Practical Labs

- **[TryHackMe](https://tryhackme.com/)**: An interactive, gamified platform that helps you learn cybersecurity through hands-on tasks and challenges
- **[Hack The Box](https://www.hackthebox.com/)**: Offers a range of penetration testing challenges in a realistic environment to enhance your hacking skills
- **[Immersive Labs](https://www.immersivelabs.com/)**: Provides on-demand labs covering various cybersecurity topics to help you develop practical skills
- **[RangeForce](https://www.rangeforce.com/)**: A cyber range platform that offers interactive training modules covering incident response, malware analysis, and other cybersecurity topics. Perfect for building technical skills with real-time feedback
- **[Blue Team Labs Online](https://blueteamlabs.online)**: A platform focused on blue team (defensive security) labs where you can practice threat detection, incident response, and forensic analysis. Ideal for those looking to develop skills for defensive roles
- **[PicoCTF](https://picoctf.org/)**: A free, game-based platform for learning cybersecurity through capture-the-flag (CTF) challenges. It’s suitable for beginners and provides valuable practice in problem-solving, binary exploitation, and web vulnerabilities
- **[PentesterLab](https://pentesterlab.com/)**: Focuses on web application security with practical exercises and hands-on experiences
- **[PortSwigger Web Security Academy](https://portswigger.net/web-security)**: A free platform offering in-depth labs for learning web security and penetration testing techniques like XSS, SQL injection, and CSRF using the Burp Suite tool
- **[Blue Team Labs Online](https://blueteamlabs.online)**: A platform focused on blue team (defensive security) labs where you can practice threat detection, incident response, and forensic analysis. Ideal for those looking to develop skills for defensive roles
- **[Practical Ethical Hacking Labs (TCM Security Academy)](https://academy.tcm-sec.com/)**: TCM Security offers a range of labs designed to teach you the fundamentals of ethical hacking and penetration testing. Their labs focus on real-world hacking scenarios and methodologies
- **[LetsDefend](https://letsdefend.io/)**: LetsDefend provides hands-on blue team training and allows you to act as a Security Operations Center (SOC) analyst. You’ll analyze incidents, investigate threats, and practice with real-time simulations
- **[AttackIQ Academy](https://academy.attackiq.com/)**: A free platform that offers labs and courses to learn how to implement threat-informed defense strategies using adversary emulation and security testing techniques. Great for those transitioning into threat hunting and adversary simulation
- **[OverTheWire](https://overthewire.org/wargames/)**: A collection of wargames that teach you security concepts in an engaging way
- **[UnderTheWire](https://underthewire.tech/wargames)**: Another set of wargames focused on security and hacking skills, suitable for various skill levels
- **[Pwn College](https://pwn.college/)**: A platform for learning about binary exploitation and web security through hands-on practice
- **[Open Security Training](https://opensecuritytraining.info/Learning%20Paths.html)**: A site offering various training paths for cybersecurity topics, including free online courses
- **[Smash The Stack](https://www.smashthestack.org/main.html)**: A platform offering a variety of labs and exercises to practice hacking and security skills
- **[SecurityBlueTeam](https://www.securityblue.team)**:  A comprehensive platform offering hands-on labs, courses, and certifications focused on blue team and defensive cybersecurity skills. SecurityBlueTeam is designed to help professionals learn how to detect, prevent, and respond to cyber threats effectively
- **[Cyber Defenders](https://cyberdefenders.org)**: his platform provides hands-on, practical cybersecurity labs that emphasize blue team skills like log analysis, network forensics, and memory forensics
- **[Pwned Labs](https://pwnedlabs.io/)**: A platform offering a variety of labs and exercises to practice hacking and security skills
